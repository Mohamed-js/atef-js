# **App Name**: Portfolio Pro

## Core Features:

- Interactive Showcase: Display projects with dynamic, animated elements to immediately grab attention.
- Skill Visualization: Utilize generative AI tool to assess the user's demonstrated abilities from his projects, and visualize skills in an engaging way.
- Contact Form with Validation: Implement a sleek contact form with real-time validation to ensure accurate submissions.
- Animated Transitions: Add smooth, polished transitions between sections for a seamless browsing experience.
- Responsive Design: Ensure the portfolio adapts flawlessly to different screen sizes and devices.
- Performance Optimization: Optimize images and code for fast loading times and optimal performance.

## Style Guidelines:

- Primary color: Deep midnight blue (#1E293B) for a sophisticated and modern feel.
- Background color: Dark gray (#2D3748) to provide a clean and unobtrusive backdrop for dark mode.
- Accent color: Electric purple (#A3A7FF) to highlight key elements and calls to action.
- Headline font: 'Space Grotesk' sans-serif for a modern, computerized feel. Body font: 'Inter' sans-serif for clean and readable text.
- Use custom, minimalist icons with subtle animations to enhance visual appeal.
- Employ a clean, grid-based layout with generous spacing to ensure readability and visual harmony.
- Incorporate smooth, subtle animations on hover and scroll events to create a dynamic and engaging user experience.
